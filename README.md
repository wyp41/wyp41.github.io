# Yupeng Wang Personal Website

Static academic personal website for Yupeng Wang.

The current PhD research section uses only a public, high-level concept figure and avoids detailed
implementation notes, unpublished data, and preliminary results. Previous-project content is based on
the resume/CV materials, the ESD paper, and the CV presentation deck.

## Local Preview

Open this file directly in a browser:

```text
I:/PhD/website/index.html
```

Or start a local static server from this directory:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Files

```text
index.html       Page content
styles.css       Visual styling and responsive layout
script.js        Year display and navigation highlighting
assets/          Website images
```
